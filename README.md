# Dotfiles

A curated collection of minimalist yet powerful dotfile configurations focused on terminal productivity and development workflow efficiency.

## Components

- [Neovim](nvim/.config/nvim/README.md) - Minimal λet functional configuration
- [WezTerm](wezterm/.config/wezterm/README.md) - Modular terminal with Vi-like navigation
- [lspmux](lspmux/.config/lspmux/README.md) - LSP multiplexer configuration

## Prerequisites

The following tools are required for optimal functionality:

- [fzf](https://github.com/junegunn/fzf) - Command-line fuzzy finder
- [fd](https://github.com/sharkdp/fd) - User-friendly alternative to find
- [rg](https://github.com/BurntSushi/ripgrep) - Fast search tool

## Font
All configurations use [Ioskeley Mono](https://github.com/ahatem/IoskeleyMono) as the primary font for consistent typography across the development environment.

## Philosophy
- Minimal and functional configurations
- Manual control over critical components
- Vi-like navigation patterns throughout
- Consistent visual aesthetic
- Performance-focused settings
